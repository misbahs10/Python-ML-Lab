# 🚢 Titanic Survival Prediction

Developed a Titanic Survival Prediction Machine Learning model using Python, Pandas, Scikit-learn, Logistic Regression, and Random Forest. Performed data preprocessing, exploratory data analysis (EDA), feature engineering, model evaluation, and achieved accurate survival predictions with a user input prediction system.

---

## 📌 Project Overview

This project uses Machine Learning algorithms to predict passenger survival based on features such as:

- Passenger Class
- Gender
- Age
- Fare
- Number of Siblings/Spouses
- Number of Parents/Children
- Embarked Port
