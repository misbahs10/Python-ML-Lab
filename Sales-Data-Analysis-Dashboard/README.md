# 📊 Sales Data Analysis Dashboard

A complete Sales Data Analysis Dashboard project built using **Python**, **Excel**, **Power BI**, and **Tableau**. This project analyzes Superstore sales data to discover business insights, sales trends, customer behavior, and regional performance through data analysis and interactive visualizations.

---

# 📌 Project Overview

The objective of this project is to analyze sales data from the Superstore dataset and answer important business questions such as:

- Which products generate the highest sales?
- Which regions perform the best?
- Which customers contribute the most revenue?
- How do sales and profit change over time?
- How does discount affect profit?
- Which product categories are most profitable?

The project includes data cleaning, preprocessing, exploratory data analysis (EDA), business analysis, and dashboard creation.

---

# 🎯 Objectives

- Clean and preprocess sales data
- Analyze monthly sales and profit trends
- Identify top-selling and most profitable products
- Analyze customer purchasing behavior
- Compare regional and category performance
- Build interactive dashboards in Power BI and Tableau
- Generate business insights for decision-making

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/misbahs10/Sales-Data-Analysis-Dashboard.git
```

Move into the project directory:

```bash
cd Sales-Data-Analysis-Dashboard
```

Create a virtual environment:

```bash
python -m venv .venv
```

Activate the virtual environment (Windows):

```bash
.venv\Scripts\activate
```

Install required libraries:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```
